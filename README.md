# Network-Security

## Environment
- Linux Ubuntu 16.04
- gcc version 5.4.0

## ● web server
- use Common Gateway Interface(CGI) 
### Build
```
git clone https://github.com/season06/Network-Security
cd web_server
make
```
## ● TLS
### Build
```
git clone https://github.com/season06/Network-Security
cd TLS
make
```
### library
-> openssl
   sudo apt-get install libssl-dev

## ● https
- Combine webserver and TLS
### Build
```
git clone https://github.com/season06/Network-Security
cd https
make
```
